# 长青伙伴 · 产品使用说明书

> 版本：v1.0  
> 适用对象：**老人（日常使用者）**、**家人（帮录入、帮设置的人）**  
> 说明：本文只讲「怎么用」，不涉及任何技术实现。

---

## 一、这是什么？

**长青伙伴**是一款帮助老人 **记得吃药、愿意吃药、吃得有记录** 的 App。

它和普通闹钟不一样：

- 进门先看到 **一棵需要照顾的小树**，不是冷冰冰的药单。
- 吃药叫 **「给小树浇水」**，吃完会被 **夸一句**，小树会随着坚持慢慢长大。
- 药单、化验单可以 **拍照存起来**，复诊时给医生看。

**适合谁用？**

| 谁 | 怎么用 |
| --- | --- |
| **老人** | 每天打开 App，看今天要浇几次水（吃几次药），吃完点一下 |
| **家人** | 偶尔帮老人 **添加/修改** 用药计划，帮设 **手机提醒**，整理 **健康档案** |

---

## 二、第一次打开，要做啥？

### 1. 设置称呼

1. 点底部 **「⚙️ 设置」**
2. 在 **「称呼」** 里填老人平时怎么叫（如：张阿姨、李爷爷）
3. 点 **「保存」**

之后首页会问候：「下午好，张阿姨」。

### 2. 添加用药计划（家人来做更合适）

如果还没有任何药：

- 首页会温和提示 **「还没有用药计划」**
- 家人可点 **「去添加药物」**，或到底部 **「💊 用药」** 添加

> **注意：** 首页故意 **不会** 用大按钮逼老人「添加新药」，避免一进来就有「我是病人」的压力。

### 3. （可选）打开「老人模式」

在 **设置** 里打开 **「老人模式」** 后：

- 底部只剩两个入口：**「🌱 我的树」** 和 **「📅 记录」**
- 用药、档案等复杂功能藏起来，避免老人误点
- 家人仍可从 **记录页底部「⚙️ 设置（家人用）」** 进入设置

## 三、首页 · 我的树（最重要）

这是老人 **每天最主要** 用的页面。

### 页面上有什么？

1. **问候语**  
   如：「下午好，张阿姨」  
   下面一句贴心话，例如：「小树今天还需要浇 2 次水（吃药）哦～」

2. **一棵小树**  
   - 显示当前形态：种子 → 小苗 → 幼树 → 小树 → 长青树  
   - **连续坚持天数**（如：🔥 连续 7 天）

3. **今日用药（按时间段分组）**  
   例如：
   - **早餐后**  
     - 降压片 · 1片 · **还没吃**  
   - **午餐后**  
     - …  
   - 当前该吃的时段旁边会有 **「现在」** 标记

4. **大按钮：「💧 一键浇水（今天都吃了）」**  
   今天所有该吃的都吃完后，按钮会变成「今天都吃完啦 💧」

### 老人怎么用？（日常流程）

**方式一：一个一个确认**

1. 到点了，打开 App  
2. 看 **「现在」** 那个时段里有什么药  
3. 吃完一种，点一下那条药  
4. 弹出 **夸夸**，点 **「好嘞！」** 关闭  
5. 状态变成 **「已吃了」**，前面出现绿色对勾  

**方式二：这一顿全吃完了**

- 点 **「一键浇水（今天都吃了）」**，今天所有还没吃的会一起记为已吃

### 重要：点了「已吃了」还能改吗？

- **可以，但要二次确认**（防手滑）  
- 再点已吃的药 → 弹窗问「要撤销吗？」→ 选 **「是的，撤销」** 才会改回「还没吃」  
- 选 **「手滑了，不撤」** 则不变  

### 小树怎么长大？

- 坚持按计划在 App 里 **打卡（浇水）**，连续天数会增加  
- 连续越久，小树形态越好（从种子长到长青树）  
- **只在 App 里打卡才算数** —— 光想着吃了、或者只在别的地方划掉提醒，小树是不会长的

## 四、用药 · 管理计划（家人常用）

点底部 **「💊 用药」** 进入。

### 能做什么？

- **手动添加** 一种新药  
- **拍照识别** 药单/包装（识别后需确认，见下文）  
- **修改** 或 **删除** 已有药物  

### 添加一种药要填什么？（尽量点选，少打字）

1. **药品名称**（唯一建议手填的地方；也可拍照识别带出）  
2. **每次剂量**：用 **「－ / ＋」** 选数量，再点 **片 / 粒 / 支** 等单位  
3. **什么时候吃**：点选卡片，可多选  
   - 早餐前 / 早餐后 / 午餐前 / 午餐后 / 晚餐前 / 晚餐后 / 睡前 / 随餐  
4. **服用周期**  
   - **长期服用**（如降压药）  
   - **短期 / 疗程**（如抗生素 7 天）→ 还要选 **一共吃几天**  
5. **从什么时候开始吃**  
   - **今天就开始** / **明天开始**  

保存后，App 会自动生成 **每天要打卡几次**（一种药选几个时段 = 几次）。

### 拍照识别怎么用？

1. 点 **「📷 拍照识别」**，对准药单或药盒拍一张  
2. 等待识别（需拍清楚、尽量少抖）  
3. 弹出 **大卡片** 显示识别结果，三个选择：  
   - **「✓ 对的，就这样」** → 进入确认页，检查后再保存  
   - **「不对，我自己改」** → 手动修改  
   - **「让家人帮我改」** → 留给家人处理  

> **重要：** 识别可能出错，**永远要以人眼确认为准**，不能拍张照就自动进药单。

### 删药时要注意

- 删除后，**今天已经打卡的记录不会消失**  
- 若曾同步过 **手机日历提醒**，删药后 **日历里的旧提醒不会自动消失**  
- 家人需到 iPhone **「日历」** 里搜索 **「小树该浇水」**，手动删掉不需要的提醒

## 五、记录 · 吃药历史与未来计划

点底部 **「📅 记录」**。

### 两个标签

| 标签 | 看什么 |
| --- | --- |
| **历史记录** | 以前哪天吃过药、月历上哪些天打了卡 |
| **未来计划** | 接下来几天，每天要在哪些时段吃哪些药 |

### 历史记录里有什么？

- **连续天数、总吃药次数、小树形态**  
- **月历**：哪天打过卡会有 ✓  
- **今天已服列表**：每种药几点打卡、哪个时段  

### 手机日历提醒（可选，家人设置）

在 **历史记录** 页有 **「手机日历浇水提醒」** 区域。

**它是干什么的？**

- 把「该打开 App 给小树浇水了」写进 **iPhone 自带日历**  
- 到点手机会响 / 弹通知，提醒老人 **打开 App 打卡**  
- **不是** 「响了就算吃完」—— 还要回 App 点「已吃了 / 浇水」  

**适合什么情况？**

- ✅ 用药 **长期稳定**、很少改  
- ❌ 经常换药、改时间 → **不建议反复同步**，否则日历里容易堆很多条，老人很难清理  

**同步前会告诉你：**

- 将添加几条提醒、每天大概几点  
- 标题类似：**「🌱 小树该浇水（早餐后）」**  

**改药之后怎么办？**

- App 里会提示 **「用药有变动，建议重新同步（慎用）」**  
- 重新同步 **不会自动删掉** 日历里旧的提醒  
- 家人需自己在 **日历 App** 里删旧的，再决定是否重新同步  

> **老人模式** 下，底部没有「设置」，可从本页底部 **「⚙️ 设置（家人用）」** 进入。

## 六、档案 · 健康单据（电子病历本）

点底部 **「📋 档案」**（老人模式下默认不显示此入口）。

### 能存什么？

- 化验单  
- 体检报告  
- 诊单 / 处方  
- 其他  

### 怎么用？

1. 点 **「📷 拍照存入档案」**  
2. 拍一张清楚的照片  
3. 选 **这是什么单据**  
4. 保存 → 按 **日期** 排好，像一本相册  

###  deliberately 不做什么？

- **不** 自动读化验数值（容易错）  
- **只存照片**，复诊时放大给医生看即可

## 七、设置

| 项目 | 作用 |
| --- | --- |
| **称呼** | 首页怎么问候老人 |
| **老人模式** | 开：只剩「我的树 + 记录」；关：显示全部功能 |
| **养成数据** | 小树形态、连续天数、总共吃了多少次 |
| **换个人用** | 以后多人用时切换账号（正式版） |

打开或关闭 **老人模式** 后，会自动回到首页，底部标签会变化。

## 八、提醒 vs 打卡 · 老人和家人必知

很多人容易混：**「手机响了」≠「已经在 App 里记为吃了」**。

| | 手机日历 / 闹钟 | 长青伙伴 App |
| --- | --- | --- |
| **作用** | 到点 **叫人** | **记录吃没吃**、**小树长大**、**看历史** |
| **完成了什么** | 提醒被划掉 | 点了「已吃了 / 浇水」 |
| **能取消提醒吗** | 在日历里自己删 | 在 App 里撤销打卡（要确认） |

**推荐跟老人说的一句话：**

> 「手机叫你是叫你 **打开长青伙伴 点一下浇水**；点完了小树才会长高。」

## 九、常见问题

**Q：今天一种药要吃三次，要打卡几次？**  
A：三次。每个时段（如早餐后、午餐后、晚餐后）各点一次。

**Q：短期药到期了还会显示吗？**  
A：不会。例如「吃 7 天」，到期后自动从今日用药和未来计划里消失。

**Q：选「明天开始」的药，今天会出现在列表里吗？**  
A：不会，到了开始那天才会出现。

**Q：老人误点「已吃了」怎么办？**  
A：再点那条 → 选「是的，撤销」。

**Q：没有用药计划时首页什么样？**  
A：显示小树 + 温和引导去添加，不会满屏药名。

**Q：不喜欢复杂功能怎么办？**  
A：打开 **老人模式**，只留树和记录；家人偶尔来帮设药即可。

**Q：换手机数据会丢吗？**  
A：正式版应支持 **备份/同步**；未开通前数据可能只在当前设备上。

## 十、给家人的快速上手清单

1. 帮老人设好 **称呼**  
2. 在 **用药** 里添加所有药（优先 **点选时间**，少打字）  
3. 若用药 **长期稳定**，在 **记录** 里 **同步一次** 日历提醒  
4. 教老人：**听到提醒 → 打开 App → 点「已吃了」→ 看夸夸**  
5. 化验单、体检报告拍进 **档案**  
6. 日常给老人开 **老人模式**，复杂页藏起来
